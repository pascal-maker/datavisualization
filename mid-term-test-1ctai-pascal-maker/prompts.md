# AI Usage Overview
 
## AI tool used
I used ChatGPT as a support tool during the development of this application.

## How it helped me
ChatGPT helped me with:
- improving and correcting my project prompt
- structuring the FastAPI backend into models, repositories, and routers
- generating example CRUD patterns for robots and functionalities
- suggesting validation checks, such as duplicate IP addresses, duplicate functionality names, blank inputs, and missing IDs
- helping structure the Gradio frontend tabs and interactions
- explaining errors and helping me debug parts of the code

## What I still did myself
I used the code from my previous assignments and asked sometimes chatgpt to help me tweak it to the requirements of this assignment. I also tested the application myself to verify that:
- the backend runs correctly
- the database entities work correctly
- robots and functionalities can be created and listed
- assignments can be set and cleared
- filtering robots by location works
- validation errors are handled correctly

## Important note
The AI tool was used as support, not as a replacement for testing or understanding the code.
I checked the final result manually to make sure the application works correctly and does not contain unverified AI-generated code.
