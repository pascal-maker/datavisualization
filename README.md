# Datavisualization

A monorepo consolidating my backend, data, and frontend projects — built with Python, FastAPI, and JavaScript.

---

## Projects

### `lab05_birds`
A FastAPI + PostgreSQL REST API for tracking bird species and spotting records. Includes Gradio-powered data visualization dashboards for exploring bird data interactively.

**Stack:** Python, FastAPI, SQLModel, PostgreSQL, Gradio, Plotly

---

### `fastapi_project`
A RESTful API for managing a university course catalog — students, lecturers, and courses. Clean router/model separation with JSON data persistence.

**Stack:** Python, FastAPI, Pydantic

---

### `chess`
A terminal-based chess engine with full board state management, piece movement logic, and a playable interface.

**Stack:** Python

---

### `merge-demo-project`
A React component playground demonstrating cart and product components with shared pricing utilities. Used to practice component composition and git merge workflows.

**Stack:** JavaScript, React

---

### `merge-demo`
A minimal HTML/CSS project used to practice and demonstrate git branching and merge conflict resolution strategies.

**Stack:** HTML, CSS

---

### `git-fundamentals`
Notes and exercises from learning core git concepts — branching, rebasing, merging, and history rewriting.

**Stack:** Markdown

---
